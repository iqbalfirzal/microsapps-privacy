Kebijakan Privasi Micros Apps

Tanggal Berlaku: 19/09/2025

Micros Apps (“Aplikasi”) adalah aplikasi resmi dari pembuat perangkat Micros (“Kami”). Aplikasi ini digunakan sebagai pendamping perangkat Micros dengan tujuan membantu pengguna dalam mengakses, mengatur, dan menganalisis data balapan.

Kami sangat menghargai privasi pengguna dan berkomitmen untuk melindungi data pribadi Anda. Dengan menggunakan aplikasi ini, Anda menyetujui kebijakan privasi berikut:

1. Informasi yang Kami Kumpulkan

- Data Perangkat: termasuk informasi teknis dari gadget Micros yang terhubung, seperti pengaturan, hasil pengukuran, dan riwayat penggunaan.

- Data Jaringan: alamat IP lokal, SSID WiFi, dan informasi koneksi yang diperlukan untuk menghubungkan aplikasi dengan perangkat Micros.

- Data Lokasi (GPS dan WiFi-based location): aplikasi mengakses data lokasi (perkiraan maupun presisi) untuk menghubungkan aplikasi dengan perangkat Micros melalui WiFi, menentukan titik start/finish balapan, serta mencatat data balapan meskipun aplikasi berjalan di latar belakang.

- Data Penggunaan Aplikasi: preferensi, riwayat sesi, dan interaksi dengan fitur aplikasi.

Kami tidak mengumpulkan data pribadi sensitif seperti nama, alamat, nomor telepon, email, atau informasi keuangan.

2. Cara Penggunaan Data

Data yang dikumpulkan digunakan hanya untuk:

  • Menghubungkan aplikasi dengan perangkat Micros.
  • Menentukan lintasan balapan berdasarkan lokasi pengguna.
  • Mencatat data balapan secara akurat, termasuk saat aplikasi berjalan di latar belakang.
  • Menyimpan riwayat balapan dan hasil pengukuran.
  • Meningkatkan kinerja aplikasi, pengalaman pengguna, serta kualitas layanan.

3. Penyimpanan dan Keamanan Data

- Data lokasi dan data lainnya tersimpan hanya secara lokal di perangkat pengguna.
- Kami tidak mengirim atau menyimpan data ke server eksternal, kecuali bila di masa depan ada layanan cloud yang akan diinformasikan secara resmi.
- Kami menerapkan standar keamanan aplikasi untuk melindungi data pengguna dari akses tidak sah.

4. Berbagi Data

Kami tidak membagikan data pengguna kepada pihak ketiga, kecuali:

  • Diperlukan oleh hukum dan peraturan yang berlaku.
  • Dengan persetujuan langsung dari pengguna.

5. Hak Pengguna

Pengguna berhak untuk:

  • Menghapus data riwayat yang tersimpan dalam aplikasi.
  • Menolak atau menghentikan penggunaan aplikasi kapan saja.

6. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini sewaktu-waktu. Perubahan akan diumumkan melalui aplikasi atau situs resmi kami.
